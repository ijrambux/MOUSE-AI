## 📖 عن المشروع

**MoUsE AI** هو API احترافي لتوليد فيديوهات بالذكاء الاصطناعي من النصوص والصور.  
يعتمد على **Agnes AI Video V2.0** ويُوفر واجهة بسيطة وقوية للمطورين.

### ✨ المميزات
- 🎥 **توليد فيديو من النص** - اكتب وصفاً وسيتم إنشاء فيديو احترافي
- 🖼️ **توليد فيديو من الصورة** - استخدم صورة كمرجع لتوجيه التوليد
- ⚡ **سريع وفعال** - متكامل مع Agnes AI Video V2.0
- 🌐 **جاهز للنشر** - يعمل على Vercel محلياً
- 📚 **توثيق تلقائي** - واجهة Swagger UI جاهزة
